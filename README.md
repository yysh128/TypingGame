# 💻 프로젝트 소개
Java를 이용한 타자연습 게임입니다.


## 프로젝트 설명
- 레벨별로 단어 떨어지는 속도 증가
- 5회 이상 틀린 단어 입력시 떨어지는 단어들의 색상이 흰색으로 변하는 페널티 적용
- 떨어지는 단어가 텍스트 입력창과 닿을때마다 하트 차감
- 하트가 다 사라질 경우 게임 종료
